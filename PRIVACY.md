# Приватность

Этот публичный билд не содержит личных сумм, имени, графика платежей, покупок или данных о питомце.

После открытия приложения все введённые данные сохраняются только в IndexedDB/localStorage конкретного браузера. Они не отправляются в GitHub и не становятся видны другим посетителям сайта.

Периодически сохраняйте резервную копию через экспорт внутри приложения.
