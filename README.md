# Мои деньги — личная PWA

Личный планировщик бюджета для iPhone. Данные хранятся локально на устройстве. Сервер и регистрация не нужны.

## Что работает

- Дашборд со свободным остатком, едой на текущую неделю и днями до зарплаты.
- Расчетный период с 5-го числа по 4-е следующего месяца.
- Карточки категорий: план, потрачено, доступно; отрицательное доступное подсвечивается красным.
- Переименование, добавление, скрытие и удаление категорий.
- Еда по четырем неделям с планом, фактом и закрытием недели.
- Накопления только в долларах.
- Платежи и остаток общей суммы.
- Запланированные покупки с проверкой, хватает ли накоплений.
- Отдельный внутренний баланс питомца без двойного списания из общего бюджета.
- Экспорт и импорт резервной копии.
- Работа офлайн после первого открытия.

## Публикация на GitHub Pages

1. Создай новый пустой репозиторий на GitHub.
2. Загрузи в корень репозитория все файлы из этой папки, включая `.github`.
3. Открой `Settings → Pages`.
4. В разделе `Build and deployment` выбери `GitHub Actions`.
5. Открой вкладку `Actions` и дождись завершения `Deploy GitHub Pages`.
6. Адрес приложения появится в `Settings → Pages`.

После публикации открой адрес на iPhone именно в Safari, нажми `Поделиться → На экран Домой → Добавить`.

## Локальная проверка

```bash
python3 -m http.server 8080
```

Затем открой `http://localhost:8080`.

Для проверки модели:

```bash
npm test
```

## Данные и резервные копии

Данные сохраняются в IndexedDB и дублируются в localStorage. Периодически используй кнопку «Скачать резервную копию» в настройках. Очистка данных Safari может удалить локальные данные приложения.
