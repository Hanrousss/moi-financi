# Самопроверка финального билда

Дата сборки: 11 июля 2026.

Проверено:

- синтаксис `app.js`, `model.js`, `storage.js` и `sw.js`;
- 7 расчетных тестов: период 5→4, июль/август, свободный остаток, перерасход, накопления, Питомец, платежи;
- валидность PWA-манифеста и наличие всех иконок;
- наличие пяти утвержденных вкладок;
- соответствие HTML-идентификаторов JavaScript-коду;
- список офлайн-ресурсов service worker;
- интерактивный smoke-тест в тестовом DOM: навигация, редактирование категории, еда по неделям, пополнение накоплений, пополнение питомца, покупки и настройки;
- архивы распаковываются, структура готова для GitHub Pages.

Ограничение проверки: установка через Safari на физический iPhone проверяется уже после публикации на GitHub Pages, поскольку PWA требует HTTPS и реальное устройство.
